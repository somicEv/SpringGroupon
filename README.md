# 简易团购项目

个人练手项目

## 项目简介：
    
    一个简单的团购网站前后台项目
    

## 项目建构：
    
    前台：
        HTML CSS jQuery
        使用FreeMarker作为模版引擎，提高加载速度
    
    后台：
        Spring MVC Spring Mybatis
        
    数据库/缓存：
        Mysql redis
    
## 优化：
    
    1、修改 -- 包名 类名
    2、添加 -- 缓存 存入Cookie
    3、添加 -- 事务管理
    4、添加 -- 前后台数据交互类 vo 用于 view层 和 controller 之间交互
    5、添加 -- 后台数据交互类 bo 用于 Controller 和 Business 之间交互

# 简易团购项目

-----个人练手项目

## 项目简介：
    
    一个简单的团购网站前后台项目
    

## 项目建构：
    
    前台：
        HTML CSS jQuery
        使用FreeMarker作为模版引擎，提高加载速度
    
    后台：
        Spring MVC Spring Mybatis
        
    数据库/缓存：
        Mysql redis
